Pure Data object to create an audio scene using spatosc.

This an object for Pure Data to create a spatosc audio scene.

Authors
-------
Mike Wozniewski <mike@mikewoz.com>
Alexandre Quessy <alexandre@quessy.net>
Tristan Matthews <le.businnessman@gmail.com>


Dependencies
------------
Requires liblo OSC library: http://liblo.sourceforge.net/
Requires spatosc.
Requires pkg-config as well.


More info
---------
Some examples and help patches are provided with this package. Please open
the patches with Pure Data.
